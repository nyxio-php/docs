# Getting started

## Installation

1. Open terminal and create project:

```shell
composer create-project nyxio/app example-app
```

2. Open created directory:

```shell
cd example-app
```

3. Run HTTP server:

```shell
php server.php
```

4. Open http://localhost:9501/api/hello

<br>
That's all you need to create and run the application.
